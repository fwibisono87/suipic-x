The tech stack for this project is:

1. Frontend: SvelteKit
2. CSS Library: Tailwind.css
3. Icon Library: Carbon
4. Fonts: 
    - Headings: Michroma
    - Body: IBM Plex Sans
    - Monospace: IBM Plex Mono
5. Backend: ElysiaJS on Bun
6. Database: PostgreSQL
7. Object Storage: Garage
8. Auth: Keycloak
